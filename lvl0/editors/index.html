Setting up your editor¶

SublimeText 3¶

There is an Imba plugin for SublimeText 3. You can install it using package control.

VisualStudio Code¶

A plugin vscode-imba can be obtained and installed from the editor's extension manager.

Atom¶

The Atom plugin repository contains a language-imba package.
